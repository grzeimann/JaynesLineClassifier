# Code of Conduct

We are committed to fostering a welcoming and inclusive community. All participants are expected to abide by this Code of Conduct.

- Be respectful and considerate.
- Avoid demeaning, discriminatory, or harassing behavior and speech.
- Tackle disagreements constructively and focus on collaboration.

If you witness or experience unacceptable behavior, please report it to the maintainers via the project issue tracker or designated contact.
