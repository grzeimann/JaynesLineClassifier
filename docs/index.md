# JaynesLineClassifier (JLC)

A Jaynesian inference engine for labeling emission-line candidates by combining spectral measurements, selection models, cosmology + luminosity‑function priors, and observed‑space rate densities.

Quick links:
- Getting started → getting-started/installation.md, getting-started/quickstart.md
- User Guide → user-guide/cli.md, user-guide/configuration.md, user-guide/examples.md
- Algorithms → algorithms/overview.md
- API → api/index.md

What’s included right now
- Phase 1: unified observed‑space rate priors integrated with measurement evidences; PPP simulator; diagnostics in outputs.
- Phase 2 (in progress): virtual volume mode, empirical fake λ‑intensity calibration, mixture fake model scaffolding, effective search measure knobs.
